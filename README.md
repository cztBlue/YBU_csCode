供盐袋寄科同学们的参考de学习笔记/作业模板

* 使用 WTFPL协议 (http://www.wtfpl.net)
* 任何人都可以自由地修改和重新发布或者宣称是自己原创都是被许可的。

我不会解决其他平台运行造成的bug。注意文件编码时UTF8，出现乱码时注意检查编译器接收的编码格式。
